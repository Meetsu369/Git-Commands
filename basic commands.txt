## ‚öôÔ∏è **Git Basics ‚Äî Setup & Info Commands**

## üìÅ **Repository Commands**

| ----------------- | ---------------------------------------------------------- | -------------------------------------------- |
| `git init`        | Initializes a new local Git repo.                          | `git init`                                   |

## üìÑ **File Tracking Commands**

| Command                   | Description                           | Example                        |
| ------------------------- | ------------------------------------- | ------------------------------ |
| `git add <file>`          | Stages a specific file.               | `git add index.html`           |
| `git add .`               | Stages all modified/untracked files.  | ‚Äî                              |
| `git commit -m "message"` | Commits staged files with a message.  | `git commit -m "added readme"` |
| `git rm <file>`           | Removes file and stages the deletion. | `git rm old.txt`               |
| `git mv <old> <new>`      | Renames or moves a file.              | `git mv old.txt new.txt`       |

## üîÑ **Branch & Merge Commands**

| Command                    | Description                             | Example                              |
| -------------------------- | --------------------------------------- | ------------------------------------ |
| `git branch`               | Lists all branches.                     | shows current branch (* highlighted) |
| `git branch <name>`        | Creates a new branch.                   | `git branch feature-1`               |
| `git checkout <branch>`    | Switches to another branch.             | `git checkout feature-1`             |
| `git checkout -b <branch>` | Creates and switches to a new branch.   | `git checkout -b dev`                |
| `git merge <branch>`       | Merges another branch into current one. | `git merge dev`                      |
